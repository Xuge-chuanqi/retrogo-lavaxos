# retrogo-lavaxos
将lavaxos移植为retrogo核心

lavaxos是由lee开源的，我这部分lava的源码参考于lee的项目。
项目地址：https://gitee.com/jacklee72/lavaxos

说明：
首先你得找一个retrogo的项目，我当前这个是在esp32和esp32-s3成功编译了的。
由于项目太大了，所以没有上传完整的项目，我只是上传了核心部分。
所以可能还是需要你们自己进行调试。
因为我只在自己的硬件上编译了，所以可能不一定适用于你们，但是核心部分应该是没有问题的。

文曲星lava游戏，需要有lavaxos文件目录

sd/lavaXOS
